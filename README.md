# 설치 및 시작
  in the main folder
  ```
  npm install
  ```
  in the Client folder
  ```
  npm install
  ```
  To start the development Client, run this command:
  ```
  npm start
  ```
  To start the development server, run this command:
  ```
  npm run dev
  ```
